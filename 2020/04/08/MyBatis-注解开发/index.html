#与$的区别

#{} 的作用主要是替换预编译语句(PrepareStatement)中的占位符?:

对于 : INSERT INTO user (name) VALUES (#{name}); ==> INSERT INTO user (name) VALUES (?);

${} 符号的作用是直接进行字符串替换:

对于 : INSERT INTO user (name) VALUES (‘${name}’); ==> INSERT INTO user (name) VALUES (‘tianshozhi’);

为了防止SQL注入，建议使用#{}.
