Lambda表达式

Lambda表达式，也可称为闭包，它是推动Java 8 发布的最重要新特性。
Lambda允许把函数作为一个方法的参数（函数作为参数传递进方法中）。使用Lambda表达式可以是代码变的更加简洁紧凑。
